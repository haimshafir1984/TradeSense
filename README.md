# TradeSense

TradeSense היא מערכת סריקת מניות חכמה עם ממשק React בעברית ושרת Express המבצע פילטור, חישוב אינדיקטורים וניקוד אסטרטגיות של משקיעים מוכרים.

## מבנה הפרויקט

- `client` - ממשק משתמש ב-React + Vite
- `server` - API ולוגיקת סריקה ב-Node.js + Express

## מקור נתונים

ברירת המחדל היא `fmp`.

כדי לעבוד עם נתוני שוק אמיתיים דרך Financial Modeling Prep:

1. מעתיקים את `.env.example` ל-`.env`
2. מגדירים `FMP_API_KEY` עם המפתח שלך מ-FMP
3. משאירים `DATA_MODE=fmp`

אם ה-API מחזיר רק חלק מהנתונים, המערכת תציג `נתונים חיים חלקיים (FMP)`.
אם מקור הנתונים כולו נכשל, השרת חוזר אוטומטית לנתוני `demo` כדי שהמערכת לא תקרוס.

## איך להריץ את המערכת

```bash
npm install
npm run dev
```
